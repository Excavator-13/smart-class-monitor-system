## 1. 修复后端登录 500 错误
- 现象：Swagger 调用 `/auth/login` 返回 500 "服务内部异常"
- 排查：health 接口显示 database "down"；用 Java JDBC 独立测试发现 `characterEncoding=utf8mb4` 抛 `Unsupported character encoding` 异常
- 根因：JDBC URL 的 `characterEncoding` 参数需用 Java 字符集名，`utf8mb4` 是 MySQL 字符集名无效
- 修复：`application.yml` 中 `characterEncoding=utf8mb4` → `characterEncoding=utf8`（Connector/J 自动映射到 utf8mb4）
- 验证：JDBC 连接成功，user 表有 admin/teacher 数据，admin123 密码 BCrypt 验证通过
- 待办：用户需重启后端生效
