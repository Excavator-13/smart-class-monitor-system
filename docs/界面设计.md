# 界面设计

## 1. 登录/注册界面

### 1.1 界面布局

``
┌─────────────────────────────────────────────────────────┐
│  [夜空视频背景 - login-moonset.mp4]                      │
│                                                         │
│                    智慧教室实时行为分析                     │
│                      与安全监测系统                       │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │                                                 │   │
│  │  [登录] / [注册]  模式切换标签                    │   │
│  │                                                 │   │
│  │  用户名: [________________]                     │   │
│  │                                                 │   │
│  │  密码:   [________________]                     │   │
│  │                                                 │   │
│  │  昵称:   [________________]  (仅注册模式)        │   │
│  │                                                 │   │
│  │  [记住密码]                                      │   │
│  │                                                 │   │
│  │              [  登  录 / 注  册  ]               │   │
│  │                                                 │   │
│  │  [AI logo - 点击5次进入开发者模式]                │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
└─────────────────────────────────────────────────────────┘
``

### 1.2 交互说明

- 背景为循环播放的夜空视频（login-moonset.mp4），视频加载失败时显示渐变背景
- 支持登录/注册两种模式切换
- 用户名和密码输入框支持回车键提交
- 登录成功后跳转至实时监控页面
- 登录失败显示错误提示（用户名或密码错误 / 用户已被禁用）
- 支持"记住密码"功能，使用 localStorage 存储 Token
- 注册成功后自动登录
- 点击 AI logo 5 次可进入开发者模式（显示调试信息）

- 支持按类型筛选：全部类型、明火检测、危险区域入侵、陌生人出现、使用手机、活体检测异常、AI换脸攻击、异常声学事件、人员摔倒、异常聚集等
- 支持关键词搜索（告警类型、视频源、人员名称）
- 每条告警显示：时间、类型、等级（带颜色标识）、视频源、状态、置信度
- 点击告警行可查看详情（截图、录像、目标信息）
- 支持告警处理：弹窗选择状态（已处理/误报/忽略）并填写备注
- 支持录像回放：点击录像按钮弹出视频播放器，自动定位到事件发生时间点
- 分页展示，每页 10 条

- 支持在视频画面上通过鼠标拖拽可视化编辑区域多边形
- 区域列表展示所有已配置区域，支持启用/禁用、编辑、删除
- 行为规则列表展示所有可配置的检测规则，支持开关和阈值调整
- 区域编辑弹窗：名称、类型、持续时间阈值、安全距离

## 10. 通用交互规范

### 10.1 导航栏

- 左侧导航栏固定，支持折叠/展开
- 当前页面高亮显示
- 导航项：实时监控、告警管理、区域规则、人脸库、用户管理（仅 admin）、系统状态
- 右上角显示当前用户名称和退出按钮

### 10.2 响应式设计

- 支持 1920x1080、1366x768、2560x1440 等常见分辨率
- 视频画面自适应容器大小
- 表格支持横向滚动

### 10.3 颜色规范

- 主色调: #1890ff / #409EFF (蓝色)
- 成功/运行: #52c41a / #67C23A (绿色)
- 警告: #faad14 / #E6A23C (橙色)
- 错误/危险/高风险: #f5222d / #F56C6C (红色)
- 信息: #1890ff (蓝色)
- 最高等级(critical): #cf1322 (深红)

### 10.4 加载状态

- 数据加载时显示 loading 动画
- 按钮操作时显示 loading 状态防止重复提交
- 视频流离线时显示"stream offline"提示画面

### 10.5 错误提示

- 网络错误: "网络连接失败，请检查网络"
- 接口错误: 显示后端返回的错误信息
- 表单验证: 输入框下方显示红色错误提示
- 视频流错误: 显示黑色画面 + 错误文字

### 10.6 开发者模式

- 在登录页点击 AI logo 5 次可进入开发者模式
- 开发者模式下显示额外的调试信息（API 响应、事件详情等）
- 通过 localStorage 持久化开发者模式状态
