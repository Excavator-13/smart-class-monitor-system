## Requirement: 重点 AI 研判助手
系统必须在实时监控页突出显示 AI 研判助手，使教师或管理员能优先看到风险指数、风险摘要、建议动作和事件时间线。

### Scenario: AI 研判助手更醒目
- GIVEN 用户位于实时监控页
- WHEN 页面展示右侧辅助面板
- THEN AI 研判助手显示为更大的重点面板
- AND 风险分数、摘要、建议动作和时间线清晰可见

## Requirement: 非实时监控页面信息密度
系统必须在告警管理、区域规则、人脸库、系统状态页面中展示足够的统计、辅助信息和状态概览，减少大面积空白。

### Scenario: 告警管理页填充信息
- GIVEN 用户进入告警管理页
- WHEN 页面渲染
- THEN 页面展示告警统计、告警列表、处置建议和证据链概览

### Scenario: 区域规则页填充信息
- GIVEN 用户进入区域规则页
- WHEN 页面渲染
- THEN 页面展示规则统计、规则编辑、区域示意、模板和下发状态

### Scenario: 人脸库页填充信息
- GIVEN 用户进入人员与人脸库页
- WHEN 页面渲染
- THEN 页面展示人员统计、人员表格、陌生人核验和注册进度

### Scenario: 系统状态页填充信息
- GIVEN 用户进入系统状态页
- WHEN 页面渲染
- THEN 页面展示视频源、服务健康、链路概览和操作日志
