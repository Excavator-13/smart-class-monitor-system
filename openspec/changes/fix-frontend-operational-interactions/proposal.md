# 修复前端操作交互与管理能力

## 背景

当前前端存在若干会影响日常值守的交互问题：登录表单不响应回车，监控工具按钮缺少可感知的动作，历史开发者身份仍可能显示在顶部，告警类型可能直接暴露英文枚举，顶部统计包含低价值信息，规则概览的区域数量未随当前视频源区域变化，视频源标签遮挡 AI 标注，管理员也尚未在前端接入已有的用户删除接口。

## 范围

- 登录表单支持按回车提交。
- “实时画面”按钮重新连接当前视频流，“区域规则”按钮提供明确的开关状态与反馈。
- 顶部身份标识只显示真实角色，不再显示开发者模式文案。
- 所有告警类型统一通过中文映射展示。
- 删除顶部“平均延迟”和“今日识别人数”卡片，并调整剩余卡片布局。
- 规则概览按当前视频源实时统计已启用区域。
- 下移视频左上角视频源标签，减少对 AI 标注的遮挡。
- 管理员通过现有 `DELETE /users/{id}` 删除其他用户，并禁止删除自己。

## 不在范围内

- 不修改 AI 日报及其相关配置。
- 不修改 AI 检测或视频绘制逻辑。
- 不新增或修改数据库表。
- 不修改 README 文件。

## 影响模块

- `frontend/src/App.vue`
- `frontend/src/styles.css`
- `frontend/src/services/smartClassApi.js`
